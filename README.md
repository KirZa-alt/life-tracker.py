🎮 Life Tracker App (Python + Streamlit)

Turn your daily life into a game-like dashboard 🚀
Track your mood, sleep, and skills while leveling up your productivity.

✨ Features
🎯 Mood tracking with progress bar
😴 Sleep tracking (hours)
📊 Skill levels (Coding, Learning, Editing, Gym)
📈 Interactive charts using Plotly
💾 Save & load data with JSON
🎮 Gamified level system
🛠️ Tech Stack
Python
Streamlit
Plotly
JSON (data storage)
🚀 Run Locally
pip install streamlit plotly pandas
streamlit run app.py
💡 Why This Project?

Instead of boring trackers, this app gamifies your life — making productivity fun and visual.
